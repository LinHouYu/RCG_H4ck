# River City Girls Trainer

一个基于 **Python + PyQt6 + pymem** 开发的《热血少女（River City Girls）》修改器，支持金币与生命值的修改与冻结。

## ✨ 功能特性
- **金币修改**：支持直接设置任意金币数值
- **金币冻结**：保持金币恒定，不会因游戏内操作而变化
- **生命值修改**：可设置当前生命值
- **生命值冻结**：保持生命值恒定（适合锁血测试）
